Single Address Lookup

Enter a Wisconsin address of your choice into the field and submit by either pressing enter, or pressing the search icon. More specific addresses have a higher chance of being found successfully.



Upon success, the address, latitude, and longitude, as well as the MURC values for that address will be displayed in the sidebar.



A marker at that address will also appear on the map. Clicking on the marker also displays the address, latitude, longitude, and MURC values.



To understand the color scheme of the map, click to expand the legend in the top left of the map.


To toggle between the map view and the table view of the results, use the buttons located on the sidebar.


The table view presents the address, latitude, longitude, and MURC values and their corresponding definitions in the form of a table.


This table can be exported as a csv file, excel file, or pdf file by clicking the corresponding buttons on the sidebar.


To input a new address, simply hit the Start Over button on the sidebar.


Drag and Drop

To use use the drag and drop feature, drag the image of the marker on the sidebar onto the map and release it somewhere within Wisconsin.


The closest address to where the marker was dropped, and the results are displayed in the same way as the single address lookup.


Upload a .CSV

Multiple MURC values for multiple address can be found at once using a .CSV file. Open the file picker by clicking the input or browse button.


Navigate to where the .CSV file is located and select it.


The .CSV file will be loaded into a table and appear over the map on the right.


Select the column containing the addresses to have their MURC values calculated. Confirmation of which column was selected will be shown above the table.


Once a column is selected, the Caluclate MURC button will appear in the sidebar. Click it to begin the calculation.


Upon completion, all addresses with MURC values found will be displayed on the map.


The table view displays the calculated values, as well as all information within the original .CSV file. The table can be exported by clicking any of the Export Table buttons.
