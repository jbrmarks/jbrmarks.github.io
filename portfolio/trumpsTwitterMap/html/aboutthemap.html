What is it?

This map was created to visualize countries based on the number of times they are mentioned in Donald Trump's Tweets. The time frame can be adjusted by moving the slider then clicking the 'Update Map' button.


How was it made?

The map makes use of an algorithm to contruct contiuous area cartograms by James A. Dougenik, Nicholas R. Chrisman, and Duane R. Niemeyer, Â©1985 by the Association of American Geographers. It also uses makes use of code to generate jenks to create the choropleth color scheme.


Metadata

The map utilizes d3's equirectangular projection. The country topojson file is a 2018 version supplied by Natural Earth, so please note that country names and borders may have changed since. This map is intented for visualization purposes only.
